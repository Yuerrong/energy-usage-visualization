# energy-usage-visualization
A small Python project demonstrating data cleaning and visualization for monthly energy usage. Created as a small portfolio example for data analytics and automation.


## What This Project Does
- Uploads monthly usage data
- Calculates total consumption for each type of energy
- Generates different types of charts using Matplotlib
- Demonstrates for data processing and visualization

## Tools and Libraries
- Python
- Google Colab
- Pandas
- Matplotlib

##  Purpose
Created a small portfolio sample to demonstrate:
- The ability to analyse datasets
- Basic automation and visualisation skills
- Good documentation and clean project structure
